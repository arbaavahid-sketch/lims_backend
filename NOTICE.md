# Tandis LIMS Internal Build Notice

This repository contains an internal deployment of SENAITE LIMS customized for
Tandis Laboratory.

The system is based on SENAITE / SENAITE.CORE, an open-source LIMS distributed
under the GNU General Public License version 2 (GPL-2.0). The original upstream
project is available at:

- https://github.com/senaite/senaite.core
- https://www.senaite.com

Local customizations in this repository and in the `senaite.core` fork include:

- Persian (`fa`, `fa_IR`) translations
- RTL layout support
- Tandis Laboratory branding and footer text
- Docker deployment wiring for the internal laboratory instance

This build is intended for internal laboratory use. If copies of this software
